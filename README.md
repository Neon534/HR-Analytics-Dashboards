# HR Analytics Dashboard | Power BI

## Project Overview

This project is an interactive **HR Analytics Dashboard** built in **Microsoft Power BI** to help organizations monitor workforce performance and make data-driven human resource decisions. The dashboard transforms raw HR data into meaningful insights through interactive visualizations and key performance indicators (KPIs).

## Features

* Employee count and workforce overview
* Gender and age distribution analysis
* Department-wise employee insights
* Employee attrition and retention analysis
* Job role and salary distribution
* Interactive filters and slicers
* Dynamic charts and KPI cards for HR reporting

## Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Interactive Data Visualization

## Key Insights

This dashboard enables HR professionals to:

* Monitor workforce demographics
* Identify attrition trends
* Evaluate department performance
* Support recruitment and retention strategies
* Make informed, data-driven HR decisions

## Purpose

The objective of this project is to demonstrate Power BI skills in data cleaning, modeling, visualization, and dashboard design while solving real-world HR analytics challenges.
